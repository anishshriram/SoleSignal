# SoleSignal
SoleSignal is a shoe embedded safety system that lets users silently trigger an emergency alert by tapping a secret pattern with their foot. When activated, it sends a realtime GPS distress message to a trusted contact, providing discreet protection when using a phone isn’t safe or possible.
